# 英语口语训练应用

一款基于**认知神经科学**的英语口语训练 Web 应用。核心理念是"像练体育一样练英语"——通过高密度重复训练，将口语练成本能，而非依赖记忆和翻译。

---

## 产品定位

本应用面向希望系统性提升英语口语能力的学习者，采用冷静、克制的设计风格（参考 Linear / Notion / Apple Fitness），避免游戏化噪音，专注于训练密度和专注感。全程无强制评分，不产生焦虑，用户对训练节奏拥有完全控制权。

---

## 训练流程（S1 → S7）

每篇文章共 9 句，按每 3 句一组分为 3 组，用户依次完成以下 7 个训练阶段：

| 阶段 | 名称 | 核心目标 |
| --- | --- | --- |
| **S1** | 整篇听感 | 盲听全文，无文本干扰，锻炼纯听力感知；支持 0.5x–1.5x 变速和进度条拖动 |
| **S2** | 标记生词 | 点击单词标记听不清的词，LLM 实时提供释义；点击句子可从该句开始播放 |
| **S3** | Baseline 自读 | 不跟原音，按自己节奏整篇朗读并录音，留存为进步起点参照 |
| **S4** | 整篇跟读 | 跟读整篇原音，全程录音；文本默认隐藏，可选显示；不评分，用身体暴露发音问题 |
| **S5** | 句级精练 | 以 3 句为一组，逐句精细练习；支持上下句导航，每句可反复录音对比 |
| **S6** | 肌肉记忆轮 | 三句连读 12 遍，形成肌肉记忆；第 10–12 遍渐进式文本遮挡（毛玻璃效果）；显示 x/12 进度 |
| **S7** | 收尾顺读 | 建议 5–7 遍三句连读，文本默认隐藏；录制 Final 录音，与 Baseline 对比感受进步 |

### 渐进式文本遮挡（S6）

S6 阶段在第 10 遍开始逐步遮挡文本，帮助用户从依赖文本过渡到纯肌肉记忆：

- 第 1–9 遍：全部文本正常显示

- 第 10 遍：第 1 句毛玻璃遮挡

- 第 11 遍：第 1、2 句毛玻璃遮挡

- 第 12 遍：三句全部遮挡

## 核心功能

### 音频系统

所有训练音频由Minimax** TTS** 生成，A/B 双角色声音（女声 Lila / 男声 Grant），采样率 24000 Hz，MP3 格式，存储于 S3 并通过 CDN 加速。每篇文章包含以下三类音频：

- 每句独立音频（`sent_xx.mp3`）

- 每组 3 句合并音频（`group_x.mp3`）

- 整篇完整音频（`full.mp3`）

### 录音策略

训练过程中的录音（S4/S5/S6）仅在本地通过 `blob URL` 即时回放，不上传服务器，保证零等待体验。只有两个关键节点的录音会异步上传至 S3：

- **S3 Baseline**：整篇自读录音，作为进步起点

- **S7 Final**：整篇顺读录音，用于与 Baseline 对比

iOS 设备通过 `audio-recorder-polyfill` 保证录音兼容性。

### 进度持久化

训练进度自动保存，中途退出后可从上次位置继续。系统记录以下数据：

- 当前训练阶段（`currentPhase`）和已到达的最远阶段（`maxReachedPhase`）

- 每句练习次数（按阶段分类统计）

- 每组训练时长和录音时长

- 总训练时长与有效输出时长（仅录音时间）

### 训练总结

每次训练结束后提供详细复盘报告，包括各阶段练习次数、总训练时长、有效录音输出时长，以及 Baseline vs Final 对比播放。

### 交互体验

- 所有播放按钮即时显示播放状态，防止用户重复点击

- 桌面端支持**空格键**快捷键控制播放/暂停

- 阶段间可自由切换，用户拥有完全控制权

- 切换阶段或翻页时，正在播放的音频自动停止

---

## 内容库

当前内置 10 篇精选训练文章，每篇 9 句，涵盖 beginner / intermediate / advanced 三个难度等级。文章内容包含重音标记（`**word**`）和停顿标记（`/`），用于指导发音节奏。

---

## 技术架构

| 层级 | 技术栈 |
| --- | --- |
| 前端 | React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui |
| 后端 | Express 4 + tRPC 11（端到端类型安全） |
| 数据库 | Drizzle ORM + MySQL（TiDB） |
| 认证 | Manus OAuth（session cookie） |
| 存储 | AWS S3（音频文件 + 用户录音） |
| TTS | Minimax

### 数据库表结构

| 表名 | 用途 |
| --- | --- |
| `users` | 用户账户（含 role 字段支持管理员权限） |
| `articles` | 训练文章元数据（标题、难度、整篇音频 URL） |
| `sentences` | 句子数据（文本、时间戳、音频 URL、分组信息） |
| `training_sessions` | 训练会话（进度、统计、Baseline/Final 录音 URL） |
| `recordings` | 录音元数据（时长、评分字段预留） |
| `group_assessments` | 组级评定结果（星级、A 类错误标记） |
