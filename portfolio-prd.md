# 温新宇（Miles）个人作品集网站 — PRD

## 一、项目概述

求职用个人作品集网站，面向 AI PM（文艺写作方向）岗位的招聘方。
目标：HR 在 30 秒内判断候选人值不值得聊。

**技术栈**：纯 HTML / CSS / JS 单文件（index.html），不用任何框架。
**部署**：Vercel 静态部署。
**语言**：纯中文。

---

## 二、站点结构

三个页面，单文件内用 hash 路由切换（#home / #projects / #writing），不刷新页面。

### 页面 1：首页（#home）

### 页面 2：项目详情页（#projects）

### 页面 3：写作详情页（#writing）

---

## 三、视觉规范

### 整体风格
Notion 式干净白底。极简、克制、无装饰。信息密度适中，留白充分。

### 字体
- 标题 / 项目名：`'Noto Sans SC', 'PingFang SC', 'Helvetica Neue', sans-serif`，font-weight: 500
- 正文 / 描述：同上，font-weight: 400
- 不使用衬线字体。整站统一无衬线。

### 色彩
- 背景：`#ffffff`
- 文字主色：`#1a1a1a`
- 文字次级：`#888888`
- 文字弱化：`#bbbbbb`
- 边框/分割线：`#f0f0f0`
- 卡片背景：`#f7f7f5`
- 按钮主色（CTA）：`#1a1a1a`（黑底白字）
- 按钮次级：透明底 + `#e0e0e0` 边框
- 强调色仅用于赞数等数据高亮：`#c45d3e`（赤陶色），极少量使用

### 间距
- 大间距（section 之间）：80px
- 中间距（元素组之间）：32-40px
- 小间距（元素内部）：12-16px
- 卡片圆角：6px
- 按钮圆角：4px

### 动画
- 所有动画保持克制，缓动函数统一 `ease-out`
- 页面切换：淡入 300ms
- hover 效果：透明度或背景色变化 200ms
- 轮播滚动：线性匀速

## 四、首页详细设计

### 4.1 导航栏（固定顶部）
- 左侧：「温新宇」（文字 logo，点击回首页）
- 右侧：「项目」「写作」两个文字链接 + 「简历下载」按钮（小号，描边样式）
- 高度：52px
- 背景：白色，底部 0.5px 分割线
- 滚动时保持固定

### 4.2 Hero 区域
- 垂直居中于首屏上半部分
- 名字：「温新宇」，22px，font-weight 500
- 副标题：「编剧，知乎体育板块成长力榜答主，前职业乒乓球运动员，球馆创业者，现任制造业北美市场产品负责人。」14px，#888
- 副标题与名字间距 8px
- 整个 Hero 块水平居中，左对齐文字，max-width 600px

### 4.3 双轨环形轮播

#### 布局
- Hero 下方，占据视口大部分面积
- 上轨道：项目（6 张卡片）
- 下轨道：写作（5 张卡片 = 4 篇知乎 + 1 个编剧作品）
- 两条轨道之间间距 24px
- 上轨道标签（左侧）：「项目 →」，11px，#bbb
- 下轨道标签（左侧）：「← 写作」，11px，#bbb

#### 轮播行为
- 上轨道：向右自动滚动
- 下轨道：向左自动滚动
- 速度：约 30px/秒，匀速线性，视觉上缓慢漂移
- 无限循环：卡片首尾相接，无缝衔接
- hover 某条轨道时：该轨道暂停，另一条继续
- 可鼠标拖拽 / 触屏滑动
- 移动端支持触摸滑动

#### 项目卡片设计
- 尺寸：宽 240px，高 170px
- 上方：截图区域（占卡片上部约 65%），背景 #f7f7f5，圆角 6px
  - 截图用 `<img>` 标签，object-fit: cover
  - 没有截图时显示项目名首字作为占位
- 下方：项目名（13px，font-weight 500）+ 一句话定位（11px，#999）
- 卡片间距：16px
- hover 效果：卡片整体上移 2px，加淡阴影 `0 2px 8px rgba(0,0,0,0.06)`
- 点击：跳转到项目详情页对应锚点

#### 写作卡片设计
- 尺寸：宽 260px，高 100px
- 纯文字，无截图
- 标题（14px，font-weight 500）
- 副信息（11px，#bbb）：日期 + 赞数/收录
- 赞数用强调色 #c45d3e
- hover 效果：背景色加深为 #f0efed
- 点击知乎文章卡片：新窗口打开知乎链接
- 点击编剧卡片：新窗口打开优酷链接（如有）或跳转项目详情页

### 4.4 底部
- 分割线
- 左侧：「© 2026 温新宇」
- 右侧：邮箱 + 电话（占位符 [邮箱] [电话]）
- 字号 12px，#bbb

#### 每个项目 section 包含（从上到下）：

1. **项目名**：22px，font-weight 500
2. **一句话定位**：14px，#888，紧跟项目名下方
3. **角色 + 日期标签**：12px，#bbb，例如「独立开发 · 2026.03 — 至今」
4. **按钮区**：水平排列

5. **封面大图/视频区**：宽度 100%，圆角 6px，背景 #f7f7f5

   - 有视频：嵌入视频播放器

## 七、交互规范

### 导航
- hash 路由切换：#home / #projects / #writing
- 浏览器前进后退正常工作
- 页面切换时淡入动画 300ms
- 当前页面对应的导航文字加粗或变为 #1a1a1a（非当前页为 #888）

### 按钮
- 主按钮（CTA）：背景 #1a1a1a，文字 #fff，padding 6px 16px，圆角 4px
  - hover：背景变 #333
- 次级按钮：背景透明，边框 0.5px #e0e0e0，文字 #888，padding 6px 16px，圆角 4px
  - hover：背景 #f7f7f5
- 按钮内文字：12px

### 响应式
- 桌面（> 768px）：正常布局
- 移动端（≤ 768px）：
  - 轮播卡片缩小：项目卡片宽 180px，写作卡片宽 200px
  - 项目详情页 padding 减小
  - 导航栏「简历下载」按钮隐藏或缩为图标

### 性能
- 图片使用 lazy loading（`loading="lazy"`）
- 轮播使用 CSS transform + requestAnimationFrame，不用 setInterval
- 减少重排：轮播容器用 `will-change: transform`
